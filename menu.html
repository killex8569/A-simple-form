Bienvenue sur le menu

Cette page web a été créée pour gérer des formulaires. Vous pourrez y retrouver de quoi créer, enregistrer, modifier et même supprimer vos formulaires.



Ressources du site

Ce site comporte normalement 14 fichiers/dossiers, dont le formulaire avec ses différentes pages. Il inclut également du code JavaScript pour vous faciliter la vie, du PHP qui vous permettra de sauvegarder vos formulaires, ainsi qu'une base de données nécessaire au fonctionnement de la sauvegarde. De plus, un framework vous permettra de naviguer plus facilement dans le menu déroulant. Si une fonction ne marche pas, ce n'est pas normal. Le site permet d'enregistrer presque toutes les données que vous avez entrées, sauf certaines comme le nom du formulaire, car cela pourrait entrer en conflit avec d'anciens formulaires enregistrés.



Comment fonctionne-t-il ?

Utiliser Mon Formulaire est simple. Il vous suffit de vous inscrire, de créer votre formulaire en ajoutant les champs dont vous avez besoin, puis de partager le lien de votre formulaire avec les personnes dont vous souhaitez recueillir les informations. Les réponses seront automatiquement collectées et organisées pour vous, vous permettant de les visualiser et de les analyser facilement.
